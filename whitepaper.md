# 巴巴博一（BBBO1）Litepaper

## 1. 專案簡介
巴巴博一是基於 Solana 的純 meme 幣，核心價值來自社群與迷因文化。專案目標是建立有趣、可參與、可流通的社群代幣，並以透明的鏈上操作建立信任。

## 2. 代幣資訊
- 名稱：巴巴博一
- 代號：BBBO1
- 鏈：Solana（SPL Token + Metaplex Metadata）
- Mint：HK9L8mMCwPxU1HtaArnPidtAwv9ubLTdpi9HsjGM2HGr
- 總量：30,000,000,000
- 小數位：7
- 供給：固定總量（關閉 mint authority）
- 可燃燒：持有人可自行燃燒代幣

## 3. 代幣分配
- 70%：流動性與公開流通（DEX 流動性池 + 市場自由流通）
- 15%：社群激勵（活動、空投、任務獎勵）
- 10%：生態/營運金庫（官網、社群維運、宣傳）
- 5%：團隊預留（匿名團隊，建議鎖倉/線性釋放）

## 4. 團隊
匿名團隊運作，專注於社群與產品推進。將透過公開資訊與可驗證鏈上操作維持透明度。

## 5. 路線圖
- 建立官網（簡單 landing page）
- 開設社群（Twitter）
- 上線 DEX
- 建立流動性
- 嘗試上 CEX

## 6. 風險與聲明
本專案為純 meme 幣，價格波動風險高，不構成任何投資建議。參與者應自行評估風險。
